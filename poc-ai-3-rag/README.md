## POC com Spring AI + RAG

Spring AI com RAG (Retrieval-Augmented Generation) permite criar aplicações inteligentes que combinam modelos de linguagem (LLMs) com dados privados e contextuais. Usando o ecossistema Spring, o RAG recupera informações relevantes de fontes como PDFs, bancos de dados ou vector stores e as injeta no prompt do modelo, reduzindo alucinações e aumentando a precisão das respostas. Essa abordagem é ideal para chatbots corporativos, busca semântica e assistentes especializados, mantendo controle, escalabilidade e integração nativa com aplicações Java/Spring.
